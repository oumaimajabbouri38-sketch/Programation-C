# 🎓 Student Grade Manager (C)

## 📌 Description
This is a simple C program that calculates the average of a student’s grades and determines if the student is admitted or failed.

---

## 🚀 Features
- Input number of subjects
- Enter grades for each subject
- Calculate average
- Display result (Admitted / Failed)

---

## 🧠 Concepts used
- Variables
- Loops (for)
- Conditions (if/else)
- Basic input/output (scanf / printf)

---

## ▶️ How to run

### 1. Compile the program:
gcc main.c -o program

### 2. Run it:
./program
